# Road Reflection Noise Removal System

### Preprocessing for Autonomous Vehicle Lane Navigation

## Overview

This repository presents a computer vision preprocessing solution designed to reduce light reflection noise on road surfaces for autonomous robotic vehicles.

Strong sunlight reflections, glare, and bright spots on the road can interfere with lane detection algorithms. These reflections may be incorrectly recognized as lane markings, causing navigation systems to make incorrect steering decisions and deviate from the center of the road.

This project focuses only on the reflection/noise removal stage before lane detection and navigation are applied.

The cleaned output generated by this system is intended to be used as an input for the main navigation and line detection pipeline available in Repository #1.

---

## Problem

In outdoor autonomous navigation, road surfaces often contain:

* Sunlight reflections
* Overexposed regions
* Bright glare areas
* High-intensity light noise

Without preprocessing, these artifacts can:

* Confuse line detection algorithms
* Create false lane candidates
* Disturb edge extraction
* Cause unstable steering behavior
* Shift the vehicle away from the road center

---

## Solution

This system applies image preprocessing techniques to suppress reflective noise before lane extraction.

The implemented approach includes:

### 1. Contrast Processing

The image contrast is adjusted to improve the distinction between actual lane markings and reflected light regions.

### 2. Reflection Area Detection

High-intensity regions likely caused by light reflection are identified.

### 3. Blur-Based Noise Suppression

Detected reflective areas are selectively blurred to reduce their influence on feature extraction and line recognition algorithms.

The resulting frame preserves road structure while minimizing false detections caused by glare.

---

## Processing Pipeline

```text
Input Camera Frame
        ↓
Contrast Enhancement
        ↓
Reflection / Bright Area Detection
        ↓
Selective Blur Filtering
        ↓
Cleaned Road Map Output
        ↓
Lane Detection & Navigation System
```
